# Project Name
USzipsProject

# Description
MongoDB project that makes statistical computations on a US zips dataset. It uses Mapbox GL JS as the API to render a map for interactivity.

# Technologies Used
- Node.js
- MongoDB
- Express.js
- HTML/CSS/JavaScript
- Mapbox GL JS
